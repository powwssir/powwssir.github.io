- 虚拟内存空间分布
  - 内核空间
  - 用户空间
    - 只读段，包括代码和常量等。
    - 数据段，包括全局变量等。
    - 堆，包括动态分配的内存，从低地址开始向上增长。
    - 文件映射段，包括动态库、共享内存等，从高地址开始向下增长。
    - 栈，包括局部变量和函数调用的上下文等。栈的大小是固定的，一般是 8 MB。

- 内存分配与回收
  - malloc() 分配内存
    - brk()
    - mmap()
  - free() / unmap() 释放内存
  - 回收内存
    - 回收缓存
    - 回收不常访问的内存，把不常用的内存通过交换分区(Swap)直接写到磁盘中
    - 杀死进程
